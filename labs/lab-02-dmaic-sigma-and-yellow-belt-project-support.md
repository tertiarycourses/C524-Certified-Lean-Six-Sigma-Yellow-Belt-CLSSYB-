# Lab 02: DMAIC, Sigma, and Yellow Belt Project Support

## Domain

Background and Lean Six Sigma Overview

## Objective

Describe DMAIC, basic sigma concepts, and the Yellow Belt role in a process improvement project.

## Tasks

1. Write the purpose of each DMAIC phase.
2. Select one improvement opportunity from Lab 01.
3. Identify how a Yellow Belt can support each DMAIC phase.
4. Define a unit, defect, and opportunity for your process.
5. Create a 20-row sample data table.
6. Count defects.
7. Calculate defect percentage.
8. Explain what DPMO and sigma level communicate.
9. Create a project support checklist for a Yellow Belt.

## Deliverable

Submit a DMAIC support map, basic measurement worksheet, and Yellow Belt checklist.

## Review Questions

1. Why is DMAIC useful?
2. What is a defect?
3. What does sigma level describe?
4. What should a Yellow Belt contribute to a project team?
