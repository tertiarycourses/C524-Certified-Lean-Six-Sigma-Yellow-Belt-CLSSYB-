# Lab 06: Value Stream Mapping and PICK Action Planning

## Domain

Process and Value Stream Mapping

## Objective

Use value stream mapping to identify improvement actions and prioritise them with a PICK chart.

## Tasks

1. Draw the current-state process flow.
2. Add wait time, process time, rework, or handoff notes where available.
3. Mark waste and bottlenecks.
4. Sketch a future-state process idea.
5. List possible improvement actions.
6. Estimate payoff for each action.
7. Estimate difficulty for each action.
8. Classify each action as Possible, Implement, Challenge, or Kill.
9. Build a simple action plan for selected items.

## Deliverable

Submit a current-state map, future-state idea, PICK chart, and action plan.

## Review Questions

1. What does a value stream map show?
2. Why compare current state and future state?
3. Which actions belong in the Implement quadrant?
4. Why should action plans include owners and dates?
