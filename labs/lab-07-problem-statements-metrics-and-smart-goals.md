# Lab 07: Problem Statements, Metrics, and SMART Goals

## Domain

Problem Solving, Risk, and Change Management

## Objective

Write a clear problem statement, select metrics, and define a SMART improvement goal.

## Tasks

1. Select one problem from earlier labs.
2. Write a factual problem statement.
3. Remove blame, assumed causes, and proposed solutions from the statement.
4. Select one primary metric.
5. Select supporting metrics for time, quality, cost, or customer satisfaction.
6. Identify the baseline.
7. Write a SMART goal.
8. Test whether the goal is specific, measurable, attainable, relevant, and time-bound.

## Deliverable

Submit a problem statement, metric list, baseline, and SMART goal.

## Review Questions

1. What makes a problem statement clear?
2. Why should the problem statement avoid solutions?
3. What makes a metric useful?
4. What makes a goal SMART?
